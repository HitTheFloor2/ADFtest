本次提交完成了一个简单的自适应距离场(ADF)画球的任务。
具体实现使用Opengl，工程可以用Visual Studio中进行查看(运行需要opengl环境)。
可执行文件位于\Debug\ADFtest.exe路径。
在具体实现的过程中，把球体的体数据从文件的形式换成了函数坐标的形式。
程序执行开始时可以输入指定的八叉树最高高度，当输入大于8时，需要等待大量时间至渲染完成。

选择了高度为6和9的两种效果截图，高度为6时，可以清晰的看见球体表面的体元密集，球体外部的体元稀疏，球体内部可见稀疏的空间。高度大于9时，球面效果变化不再明显。
